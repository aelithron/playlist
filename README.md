## Features
- Create playlists, either from scratch or from an existing playlist on another platform.
- Generate beautiful, aesthetically pleasing "share pages" for your playlists.
- Export playlists to any of the supported platforms.
- "Quick Switch" to transfer playlists directly between platforms.
### Supported Platforms
- [TIDAL](https://tidal.com)
- [Spotify](https://spotify.com)
- [YouTube Music](https://music.youtube.com)
- [Jellyfin Server](https://jellyfin.org)
  - Note: Your server must be [accessible remotely](https://jellyfin.org/docs/general/post-install/networking) over the Internet.
- [Amazon Music](https://music.amazon.com)
- [Pandora](https://pandora.com)
